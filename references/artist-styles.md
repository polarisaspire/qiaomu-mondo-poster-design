# Mondo Artist Style Variations

Different Mondo artists have distinctive visual languages. Use these style modifiers to target specific aesthetics.

## Table of Contents
- Tyler Stout - Maximalist Character Collages
- Olly Moss - Minimalist Negative Space Master
- Martin Ansin - Art Deco Elegance
- Laurent Durieux - Visual Puns & Symbolism
- Jay Ryan - Single Image Distillation
- Kilian Eng - Geometric Futurism
- Drew Struzan - Classic Painted Realism

---

## Tyler Stout Style

**Characteristics:**
- Dense, character-packed compositions
- Intricate line work with multiple focal points
- Maximalist approach - every inch filled with detail
- Comic book influence
- High contrast black linework
- Organized chaos - busy but readable

**Prompt Modifiers:**
```
Tyler Stout style Mondo poster, densely layered composition, character collage,
intricate line work, multiple characters integrated into single scene, high
contrast black ink lines, organized maximalist design, comic book influenced,
screen print with fine detail
```

**Best For:** Ensemble films, franchises, epic stories

**Example:**
```
Star Wars Empire Strikes Back in Tyler Stout style, densely packed character
collage centered on Yoda and Luke training, AT-AT walkers, Cloud City, Darth
Vader looming large, intricate black line work, 4-color screen print: black,
white, red accent, blue, organized chaos composition, every space filled with
iconic moments, vintage 1980 poster
```

---

## Olly Moss Style

**Characteristics:**
- Ultra-minimalist compositions
- Clever negative space usage
- Visual double meanings (hidden imagery)
- 1-3 color palettes
- Simple geometric shapes
- Iconic objects over characters
- "Less is more" philosophy

**Prompt Modifiers:**
```
Olly Moss style Mondo poster, ultra minimalist design, clever negative space,
hidden imagery within silhouette, 2-color limited palette, geometric simplicity,
visual double meaning, symbolic object-focused, extremely clean composition
```

**Best For:** Films with iconic imagery, conceptual themes, noir

**Example:**
```
Jaws in Olly Moss style, minimalist shark fin that doubles as beach umbrella
silhouette, 2-color print: ocean blue and sunset orange, ultra clean negative
space, geometric simplicity, symbolic visual pun, no characters shown, retro
1975 minimalist poster
```

## Martin Ansin Style

**Characteristics:**
- Art Deco influence
- Elegant, flowing line work
- Muted vintage color palettes
- Sophisticated composition
- Architectural elements
- Period-accurate styling
- Refined and polished

**Prompt Modifiers:**
```
Martin Ansin style Mondo poster, Art Deco influence, elegant flowing lines,
muted vintage palette, sophisticated composition, architectural elements,
refined line work, period styling, polished screen print aesthetic
```

**Best For:** Classic Hollywood, period pieces, gothic horror

**Example:**
```
Dracula in Martin Ansin style, Art Deco gothic castle with flowing cape silhouette,
elegant curved lines, muted palette: burgundy, gold, cream, black, sophisticated
symmetrical composition, architectural ornate details, refined vintage horror
aesthetic, 1931 Universal Monsters poster styling
```

## Laurent Durieux Style

**Characteristics:**
- Visual puns and hidden meanings
- Atmospheric lighting effects
- Subtle symbolism
- Gradients and tonal shifts (within screen print limits)
- Mysterious and evocative
- Smart composition tricks

**Prompt Modifiers:**
```
Laurent Durieux style Mondo poster, visual pun composition, hidden symbolic
imagery, atmospheric lighting, subtle gradient within screen print style,
clever composition trick, mysterious evocative mood
```

**Best For:** Thrillers, mysteries, films with twists

**Example:**
```
Inception in Laurent Durieux style, city folding onto itself hidden within
business man's briefcase opening, visual pun, atmospheric blue-gray gradient,
4-color screen print: navy, gray, bronze, cream, clever symbolic composition,
mysterious dreamlike mood, vintage 1970s psychological thriller aesthetic
```

---

## Jay Ryan Style

**Characteristics:**
- Single representative image
- Focus on one symbolic element
- Bold, simple shapes
- Textured backgrounds
- Handmade feel
- Folksy charm
- Limited but warm palettes

**Prompt Modifiers:**
```
Jay Ryan style Mondo poster, single focal image, bold simple shapes, textured
hand-printed background, folksy handmade aesthetic, warm limited palette,
one symbolic element represents entire film
```

**Best For:** Coming-of-age films, indie dramas, nostalgic stories

**Example:**
```
Stand By Me in Jay Ryan style, single image of railroad tracks vanishing into
forest, bold simple composition, textured paper background, warm earth tones:
rust, cream, forest green, handmade screen print feel, folksy nostalgic charm,
vintage 1980s indie film poster
```

---

## Kilian Eng Style

**Characteristics:**
- Geometric futurism
- Precise technical line work
- Sci-fi architecture focus
- Cool color palettes (blues, purples, teals)
- Isometric or angular perspectives
- Digital precision meets analog printing
- Atmospheric depth

**Prompt Modifiers:**
```
Kilian Eng style Mondo poster, geometric futuristic design, precise technical
line work, sci-fi architectural elements, cool teal and purple palette,
angular perspective, clean digital precision in screen print, atmospheric depth
```

**Best For:** Sci-fi, cyberpunk, futuristic settings

**Example:**
```
Blade Runner in Kilian Eng style, geometric isometric city scape with flying
cars, precise architectural line work, 3-color print: electric blue, neon pink,
dark teal, angular perspective, futuristic technical precision, atmospheric
neon-lit depth, retro 1982 cyberpunk aesthetic
```

---

## Drew Struzan Style

**Characteristics:**
- Painted realism (adapted for screen print)
- Epic cinematic compositions
- Character portraits with personality
- Warm nostalgic glow
- Classic Hollywood poster tradition
- Multiple vignettes integrated
- Dramatic lighting

**Prompt Modifiers:**
```
Drew Struzan style Mondo poster, painted realism adapted for screen print,
epic cinematic composition, character portraits integrated, warm nostalgic
lighting, classic Hollywood poster tradition, dramatic staged lighting,
multiple scene vignettes
```

**Best For:** Adventure films, blockbusters, ensemble casts

**Example:**
```
Indiana Jones in Drew Struzan style, hero portrait with fedora and whip,
painted realism screen print, multiple adventure vignettes surrounding central
figure, warm sunset glow, 6-color print: earth tones with gold highlights,
dramatic cinematic lighting, classic 1981 adventure poster aesthetic
```

## Mixing Artist Styles

You can combine elements from multiple artists:

```
[Film] in Mondo poster style, [Artist A]'s [characteristic] meets [Artist B]'s
[characteristic], [shared visual elements], [combined color approach]
```

**Example:**
```
Alien in Mondo poster style, Olly Moss's minimalism meets Kilian Eng's geometric
technical precision, single xenomorph silhouette constructed from architectural
spaceship elements, 2-color print: industrial gray and bio-organic green,
ultra-clean negative space with precise technical line work
```

## Choosing the Right Artist Style

- **Dense, detailed, fanboy posters** → Tyler Stout
- **Clever, minimal, conceptual** → Olly Moss
- **Elegant, refined, classic** → Martin Ansin
- **Mysterious, atmospheric, smart** → Laurent Durieux
- **Warm, simple, heartfelt** → Jay Ryan
- **Futuristic, technical, geometric** → Kilian Eng
- **Epic, painted, cinematic** → Drew Struzan
